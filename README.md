## Описание проекта

В черте населённых пунктов на Камчатском полуострове люди часто сталкиваются с бурыми медведями. Нехватка естественной пищи в неурожайные годы, массовая прикормка, неправильная утилизация рыбных отходов и сокращение научного мониторинга поведения животных способствуют появлению медведей разных районах Петропавловска-Камчатского и других населённых пунктах края.
Суть проекта заключается в детекции медведей на фотоизображениях (с фотоловушек, камер видеонаблюдения) для возможной передачи заинтересованным подразделениям органов власти с целью обеспечения безопасности местных жителей (информирование граждан о зафиксированных животных, адресный выезд оперативных групп).

## Инструкция по установке и использованию

1. Откройте файл с кодом (расширение `.ipynb`) на своём компьютере.

2. Убедитесь, что установлены все необходимые библиотеки. Если нет, то запустите следующую команду в первой ячейке файла:

    !pip install inference-sdk

3. Запустите вторую ячейку кода с импортами необходимых библиотек.

   from inference_sdk import InferenceHTTPClient
   
   import json
  
5. Внесите необходимые изменения в третью ячейку кода.

4.1. Найдите блок кода "# 3. Инициализация клиента в Roboflow", в нём строку api_key="_____". Вместо символов подчёркивания вставьте уникальный API-ключ, полученный из Вашей учетной записи.

    # 3. Инициализация клиента в Roboflow
    ...
        api_key="_____" # <- Вставьте сюда Ваш API-ключ

4.2. Найдите блок кода "# 4. Запускаем рабочий процесс для изображения", в нём строку "image": "_____.jpg". Вместо символов подчёркивания вставьте ссылку на Ваше изображение.

    # 4. Запускаем рабочий процесс для изображения
    ...
        "image": "_____.jpg" # <- Вставьте сюда ссылку на Ваше изображение

5. Выполнение.

5.1. Запустите все ячейки файла последовательно.

5.2. Искомый результат будет отображён в конце третьей ячейки (например, "На изображении обнаружено медведей: 1").
     Теперь данные можно направлять в заинтересованные ведомства с координатами фотоловушек либо камер видеонаблюдения.
     
5.3. Общий результат по изображению выведен в конце четвёртой ячейки.

## Дополнительная информация по изображениям

Для тестирования использовались изображения общего пользования, скачанные на локальную машину. Прикладываю файлы исходные и обработанные в Roboflow.
